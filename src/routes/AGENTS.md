# routes

## Purpose
SvelteKit file-based routing directory for defining pages, layouts, and endpoints.

## Key Files
| File | Description |
|------|-------------|
| `+layout.svelte` | Root layout component |
| `+page.svelte` | Root page component |
| `+error.svelte` | Root error boundary |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `api/` | API routes and endpoints |

## For AI Agents
