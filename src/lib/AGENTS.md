# lib

## Purpose

Contains reusable application logic, state definitions, individual "apps" for the iPhone clone, and utilities.

## Subdirectories

| Directory | Purpose                                                  |
| --------- | -------------------------------------------------------- |
| `apps/`   | Individual iPhone applications (Calculator, Music, etc.) |
| `config/` | Database and API configurations (Supabase, Turso)        |
| `server/` | Server-only utilities                                    |
| `states/` | Global Svelte Runes states (`.svelte.ts`)                |
| `types/`  | TypeScript type definitions                              |
| `ui/`     | Shared UI components                                     |
| `utils/`  | Utility functions and permissions                        |
